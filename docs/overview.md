# AI株分析メディア構築プロジェクト — 全体構想

## 概要

GPTを用いた株分析を資産化するための**自動ブログ生成 + SNS連携システム**。

分析結果を構造化データとして保存し、ブログ記事とX投稿を自動生成する。

```
分析 → 構造化 → ブログ生成 → SNS投稿 → 反応分析
```

単なるブログではなく**株分析データベース型メディア**を目指す。

---

## 目的

### 1. 分析内容の資産化

現在行っている株分析を単発の情報ではなく**データとして蓄積**する。

将来的に構築するナレッジベース：
- 銘柄別分析履歴
- 投資判断の変化
- セクター分析

### 2. コンテンツ自動生成

分析データを元に以下を自動生成：
- ブログ記事
- X投稿
- Xスレッド

### 3. SNSによる集客

```
X → ブログ
```

この導線を自動化する。

---

## 想定フロー

| ステップ | 内容 | 詳細 |
|---|---|---|
| STEP1 | 分析 | GPTsで銘柄分析（企業概要・投資仮説・成長性・リスク・バリュエーション） |
| STEP2 | データ構造化 | 分析結果をJSON形式に変換 |
| STEP3 | DB保存 | WordPress（MySQL）へ保存 |
| STEP4 | 記事生成 | 保存データからブログ記事を自動生成 |
| STEP5 | X投稿生成 | 同データからSNS向け短文を生成 |
| STEP6 | 投稿 | ブログ公開 + X投稿 |

---

## システムの特徴

### データベース型ブログ

通常のブログが「記事を書く」のに対し、このシステムは**分析データが主**でコンテンツは副産物。

### 再利用可能

1つの分析データから以下を生成できる：
- ブログ記事
- X投稿
- Xスレッド
- 銘柄ページ（将来）
